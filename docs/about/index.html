About Leschnitz Micro Actions

An artistic exploration of Upper Silesia

Project Overview

The Leschnitz Micro Actions project is an ongoing artistic research initiative that transforms local civic data from Upper Silesia (Oberschlesien/Górny Śląsk) into poetic micro-interventions. Updated every 3 hours, this project creates a continuous stream of artistic responses to the cultural and administrative landscape of this historically complex region.

Historical Context

Upper Silesia (Oberschlesien in German, Górny Śląsk in Polish) is a region with over 1000 years of documented history. Key historical periods include:

  • Medieval Period: Part of the Polish Piast dynasty, later Bohemian Crown
  • 1526-1742: Habsburg rule under Austria
  • 1742-1945: Prussian and later German administration
  • 1945: Crucial turning point - region becomes part of Poland
  • Post-1945: Population shifts, with German inhabitants expelled or fleeing
  • Present: Part of modern Poland, EU member since 2004

Methodology

The project employs automated processes to:

  1. Aggregate local news and civic data from RSS feeds
  2. Filter content for relevance to the region
  3. Transform data through AI-powered artistic interpretation
  4. Generate poetic "micro actions" - small artistic interventions
  5. Publish continuously to this website via GitHub Pages

Each micro action represents a creative response to contemporary events, filtered through the lens of the region's complex cultural heritage.

Artistic Approach

The micro actions explore themes including:

  • Post-war identity and displacement
  • Linguistic heritage and multilingualism
  • Industrial transformation and environmental change
  • Bureaucratic aesthetics and civic participation
  • Memory, nostalgia, and cultural preservation
  • Border regions and transnational identity

Technical Implementation

This project runs entirely on open-source technologies:

  • Python pipeline for data processing
  • AI language models for content generation
  • GitHub Actions for automated updates
  • Static site generation via GitHub Pages
  • No tracking, cookies, or user data collection

Cultural Sensitivity

We acknowledge the complex and sometimes painful history of Upper Silesia. This project aims to:

  • Respect all cultural perspectives - Polish, German, Czech, and Silesian
  • Use historically accurate place names in appropriate contexts
  • Avoid political bias while acknowledging historical facts
  • Create space for artistic interpretation of contemporary life
  • Honor the memory of all who have called this region home

Data Sources

Content is aggregated from public RSS feeds including:

  • Local government websites (gmina/powiat)
  • Regional news outlets
  • Cultural institution announcements
  • Community bulletin boards

All source links are preserved and attributed in each micro action.
