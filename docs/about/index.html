Each micro-action is a small rebellion against organized forgetting—a fragment of dignity restored to those displaced since 1945.

Chronology of Erasure

History written by Expansionists always has the same plot—they arrive to "civilize" empty land:

  • Before 1945: Complex tapestry of languages, religions, cultures
  • 1945-1946: Mass deportation disguised as "repatriation"
  • 1946-1950: Systematic erasure of German inscriptions, gravestones, signs
  • 1950-1989: Enforced amnesia—speaking German could cost your job
  • 1989-2004: Cautious "reconciliation" that never names the crime
  • 2004-Present: EU membership somehow makes ethnic cleansing "historical"

We reject this timeline. Every moment since 1945 is Year Zero plus X days of denial.

Our Methodology of Resistance

We perform data archaeology on the Occupier's own propaganda:

  1. Harvest civic announcements that pretend history began in 1945
  2. Detect absences—what names are never mentioned? What dates avoided?
  3. Transform bureaucratic violence into sensory experiences
  4. Generate micro-actions that make invisible marginalization visible
  5. Accumulate evidence of erasure in public view

Each action is a small incision in the smooth surface of official narrative. Death by a thousand paper cuts—using their own documents against them.

Tactics of Revelation

Our micro-actions expose hidden propaganda through:

  • Mapping absences—every missing gravestone is a data point
  • Recording silence—what languages don't exist in schools?
  • Smelling erasure—how does ethnic cleansing affect bread recipes?
  • Touching denial—why do certain walls sweat in certain weather?
  • Tasting displacement—what flavors vanished with the populations?
  • Hearing ghosts—whose footsteps echo in "empty" buildings?

Technical Implementation

This project runs entirely on open-source technologies:

  • Python pipeline for data processing
  • AI language models for content generation
  • GitHub Actions for automated updates
  • Static site generation via GitHub Pages
  • No tracking, cookies, or user data collection

Against Forgetting

We refuse the comfortable lie of "reconciliation" that erases accountability:

  • We use German names because they were erased
  • We document absence because presence is propaganda
  • We smell, taste, touch history because statistics lie
  • We create actions, not monuments—process, not product
  • We honor the displaced by refusing to let Settlers sleep peacefully

This is not neutral. Memory is not neutral. Forgetting is violence.

Data Sources

Content is aggregated from public RSS feeds including:

  • Local government websites (gmina/powiat)
  • Regional news outlets
  • Cultural institution announcements
  • Community bulletin boards

All source links are preserved and attributed in each micro action.
